Koth basic commands

For these commands you need the permission: koth.admin.edit

The koth editor commands are:
  • asmember
  • list
  • version
  • reload
  • tp
  • info
  • ignore
  • next

/koth asmember Permission: koth.admin.help Default: Operators

Shows the help menu as a normal player would see it.

/koth list Permission: koth.list Default: Operators

Shows a list of all available KoTH's

/koth version Permission: koth.version Default: True

Shows the version number, author and a link to the thread

You can disable this feature, but for the people that don't, much appreciated :)

/koth reload Permission: koth.admin.reload Default: Operators

This command reloads the entire plugin. All configs, json files, everything. Pretty helpful when you make minor changes.

This does stop all running KoTHs!

/koth tp <koth> [area] Permission: koth.admin.tp Default: Operators

With the tp command you can easily teleport to the koths. Very helpful when you don't want to remember the coords, or just forgot where it was! This teleports you to the middle of the first defined area. You can also teleport to a specific area by using the optional [area] argument.

To make sure you don't teleport in the ground, it will teleport you to the highest block. Keep that in mind if it's underground!

/koth info Permission: koth.admin.info Default: Operators

The info command is quite big, so I'm going to split it up in 3 sections, but I just want to tell you that it's currently my most favorite command. The info command will give you all the relevant information you need.

/koth info koth <koth>

The koth info command will show you the following information:

  • Name
  • Last winner
  • Location
  • Linked loot
  • Loot position
  • Areas
  • Schedules linked

/koth info loot <loot>

The loot info command will show you the following information:

  • Name
  • Koths linked
  • Schedules linked

/koth info schedule <schedule>

The schedule info command will show you the following information:

  • ID
  • Day
  • Time
  • KoTH
  • Loot
  • Capture type
  • Capture time
  • Max runtime
  • Time before it starts

/koth ignore Permission: koth.ignore Default: True

Ignores the global messages of the KoTH, this is for the players that might find the messages a bit spammy.

/koth next Permission: koth.next Default: True

Shows which KoTH is scheduled next.
