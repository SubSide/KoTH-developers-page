Loot commands

For these commands you need the permission: koth.admin.loot

The loot commands are:

  • create
  • edit
  • remove
  • rename
  • list
  • asmember
  • commands

/koth loot create <loot>

This will create a new editable, linkable loot chest with the given name.

/koth loot edit <loot>

This will open the loot chest that you can edit.

/koth loot remove <loot>

Removes the loot chest.

/koth loot rename <loot> <newname>

Renames the loot chest.

/koth loot list

Shows a list of all created loot chests.

/koth loot asmember [...args]

Members see the /koth loot differently as an Administrator, Now to make sure you know what members actually see you can use this command.

/koth loot cmd <loot>

Loot commands is a great way to give players a reward that can only be given through commands

WARNING! KoTH is executing commands with NO limitations.
For safety, at least make sure that only OP's can access this command

/koth loot cmd <loot> add <command>

Add a command to the loot commands list

/koth loot cmd <loot> list

List all commands with their ID in the loot commands list

/koth loot cmd <loot> remove <id>

Remove a specific command in the list with the given #ID
