Koth editor commands

For these commands you need the permission: koth.admin.edit

The koth editor commands are:
  • area
  • loot
  • rename

/koth edit <koth> area

With the area command you can define and remove new areas. Meaning you can have multiple areas a player could stand in to capture.

A nice way to see it is to imagine two towers. Now the player can go in one of the towers to cap it. The other players have to guess in which tower he is to actually get him off the capture zone. Now imagine 20 capture zones! You could make like a maze! :P

There is not really a limit on how many areas you can select, the servers performance is your limit!

When a koth is running, you could edit the areas, and it would take immediate effect!

/koth edit <koth> area list

Shows a list with all the defined areas.

/koth edit <koth> area create <area>

This command creates new capture zones with the given name for players to stand in. Just like with creating the koth in the first place, you must have an area selected with worldedit.

/koth edit <koth> area edit <area>

This command will overwrite previous defined areas.

/koth edit <koth> area remove <area>

Removes a previous defined area.

/koth edit <koth> loot

/koth edit <koth> loot setpos

Sets the position of where the loot chest should spawn.

/koth edit <koth> loot link <loot>

Links a certain loot chest to the koth.

If no loot chest is set, it will try to grab the default chest from the config. If no chest has been set, no loot will spawn.

/koth edit <koth> rename <name>

Renames the koth area.

Make sure you update your schedules!
