Basic editor commands

The basic editor commands are:

I'm not going too deep in to those commands. As they are pretty self explanatory.

  • create
  • remove

/koth create <koth> Permission: koth.admin.create Default: Operators

Creates a new koth area. To do this you need to have a worldedit area selected. When created it will automatically create an area with the name of the koth.

/koth remove <koth> Permission: koth.admin.remove Default: Operators

This will remove a koth. Not much else.
